# chara-convert

AI Character Card Conversion Workbench

Convert and edit AI character cards between platforms.

## Usage

```bash
# CLI
uv run ccv convert my_char.png --target janitorai --out report.md
uv run ccv diff my_char.json --target sillytavern_v2
uv run ccv list-platforms

# Web UI
uv run ccv webui
```

## Supported Platforms

- SillyTavern V2
- Janitor AI
- Backyard AI (Faraday)
- RisuAI
- Agnai
